# B2B Market

Платформа для поиска производителей, поставщиков и контрактных производств.

---

# Миссия

Создать крупнейшую B2B-платформу для предпринимателей, которая объединяет:

- производителей
- поставщиков
- контрактные производства
- импортеров
- аналитические сервисы
- образовательную базу знаний

Главная цель — помочь предпринимателю пройти путь:

Идея → Поиск ниши → Поиск производителя → Закупка → Продажа.

---

# MVP (Этап 1)

Первая версия продукта будет представлять собой агрегатор производителей.

На данном этапе компании не добавляют себя самостоятельно.

Данные собираются автоматически при помощи парсеров.

Источники данных:

- российские каталоги производителей
- сайты контрактных производств
- открытые B2B-каталоги
- выставки
- государственные реестры
- китайские площадки
- открытые данные
